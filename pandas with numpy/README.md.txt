# 📊 Pandas + NumPy Practice

A comprehensive set of exercises and examples combining **Pandas** and **NumPy** for data analysis. This repository is designed to help learners understand how to clean, manipulate, and analyze datasets using Python’s most popular data libraries.

---

## 📌 Project Overview
This repository covers:
- **NumPy basics** for numerical operations.
- **Pandas fundamentals** for tabular data handling.
- **Missing data handling** (drop, fill, interpolation).
- **Merging and joining datasets**.
- **GroupBy operations** for split–apply–combine.
- **Aggregation techniques** for statistical summaries.
